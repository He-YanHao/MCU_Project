# 引脚

## 分布及作用

| 引脚  | 作用                                      |      |      |
| ----- | ----------------------------------------- | ---- | ---- |
| GND   | 电源地                                    |      |      |
| VCC   | 电源正                                    |      |      |
| NC    | 保留                                      |      |      |
| PEN   |                                           |      |      |
| BUSY  | 没接                                      |      |      |
| T-CS  | 触摸屏片选信号，低电平使能                |      |      |
| SD-CS | 液晶屏片选信号，低电平使能                |      |      |
| DI    | SPI总线写数据信号                         |      |      |
| CLK   | 时钟                                      |      |      |
| DO    | SPI总线读数据信号，如无需读取功能则可不接 |      |      |

| 引脚编号  | 作用                                 |      |
| --------- | ------------------------------------ | ---- |
| P00 ~ P07 | 数据                                 |      |
| P10 ~ P17 | 数据                                 |      |
| P25       | TFT_WR / 写入数据时激活              |      |
| P26       | TFT_RS / 命令数据选择                |      |
| P27       | TFT_CS / SPI片选                     |      |
| P32       | TFT_RD / 低从LCD读取数据，默认高电平 |      |
| P33       | TFT_RST / 重启                       |      |
| P34       | 没接                                 |      |

P00~P07 = PA0~PA7
P10~P17 = PD8~PD15
P25 TFT_WR = D0   写入数据时激活
P26 TFT_RS = D1   命令数据选择
P27 TFT_CS = D4   SPI片选
P33 TFT_RST = D6  重启
P32 TFT_RD = D7   低从LCD读取数据，默认高电平
