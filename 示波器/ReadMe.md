# 示波器

## 项目说明

基于STM32f407的示波器。

使用DMA搬运两个包含135个成员的uint8_t数据类型的数组ADC_Num1 ADC_Num2

流程：

1. ADC_Num1接收数据
2. 使用ADC_Num2的数据清除上一次图像
3. 使用ADC_Num1的数据制作这一次图像
4. 搬运数据ADC_Num1-->ADC_Num2
5. 循环
